El TP esta compuesto por dos archivos, uno llamado grafo.py el cual tiene la implementacion de un 
objeto grafo y uno llamado tp.py el cual es el tp general que realiza todas las tareas solicitadas,
es decir, el ejecutable.

El TP fue probado en equipos con linux, utilizando python 2, y requiere de la instalacion de gnu.plot.

Para ejecutar el tp, el main cuenta con un parser de argumentos (arg.parse), el cual mediante la 
bandera -h imprimira las opciones disponibles. Esta version no cuenta con un grafo por defenco por
lo que sera necesario pasar uno como argumento (para mas imformacion del formato del grafo ver
"parserGrafo" en grafo.py).
